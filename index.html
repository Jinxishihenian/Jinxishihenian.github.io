初遇hexo

發表於 2021-03-23 更新於 2021-04-01

起因

本来想着搭个博客,了解到github+hexo建站的形式,看了下了hexo的主题还不错就决定入手,这里记录下常见命令

常见命令

命令 作用
hexo new “hellow-hexo” 新建文章
hexo generate 生效更改
hexo deploy 将生成的更改部署到码云或者GitHub上
hexo clean 清除已经生成静态文档

常见操作

操作 命令组合
发布文章 hexo clean;hexo g;hexo d
删除文章 hexo d -g
新建文章 hexo new “文章名称”
